WHAT USER NEEDS TO DO NEXT
================================================================================

PHASE 1: Install Prerequisites (One-Time, 30 minutes)
  1. Windows Package Manager installations:
     - PostgreSQL 16
     - Java 21
     - Node.js LTS
     - Python 3.12
     - pnpm

  2. Download and install:
     - Redis 7 (MSI or zip)
     - OPA binary (opa.exe)

PHASE 2: Set Up Database (One-Time, 5 minutes)
  1. Create PostgreSQL user and database
  2. Grant privileges to orionops user

PHASE 3: Start Local Services (Development, 2 minutes)
  Terminal 1: redis-server
  Terminal 2: opa run --server --addr=0.0.0.0:8181 infra/opa/policies/
  Terminal 3: cd backend && ./mvnw spring-boot:run
  Terminal 4: pnpm dev:web

  Web loads at http://localhost:3000

PHASE 4: Deploy Cloud Services (Optional, 1-2 hours)
  1. Create Render account
  2. Deploy Keycloak, AI Service, MinIO
  3. Set up Bonsai OpenSearch and Upstash Kafka
  4. Update backend environment variables
  5. Configure GitHub Actions secrets for heartbeat

PHASE 5: Verify Setup
  1. Health check all local services
  2. Health check Render services
  3. Test web login
  4. Verify GitHub Actions heartbeat

See QUICK_START.md for fast setup or SETUP_NO_DOCKER.md for complete guide.
